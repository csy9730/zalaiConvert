# rk3399pro


RK3399Pro

- Dual-core Cortex-A72 up to 1.8GHz
- Quad-core Cortex-A53 up to 1.4GHz
- NPU up to 3.0TOPS
- Mali-T860MP4 GPU
- Dual-channel DDR3/DDR3L/LPDDR3/LPDDR4
- 4K UHD H265/H264/VP9
- HDR10/HLG
- H264 encoder
- Dual MIPI CSI and ISP
- USB Type-C
